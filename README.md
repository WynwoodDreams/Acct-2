# Kept — Personal Accountability Tracker

A clean, modern accountability app that helps you commit to actions across all areas of life and honestly track whether you follow through.

## Features

- **Profile Setup** — Strengths, improvement areas, work/life context
- **Accountability Dashboard** — Commitments with deadlines, auto check-ins, follow-through tracking
- **AI Suggestions** — Personalized commitment recommendations based on your profile
- **Calendar & Scheduling** — One-time, daily, weekdays, custom days, or specific dates
- **Browser Notifications** — Alerts when commitments are due
- **7-Day History** — Visual streak tracking per commitment

## Tech Stack

- React 18 + Vite
- Zero dependencies beyond React
- localStorage persistence
- Mobile-first responsive design

## Getting Started

```bash
npm install
npm run dev
```
